🎨 Сайт-рисовалка

Сайт-рисовалка - это веб-приложение для создания рисунков прямо в браузере. Проект позволяет пользователю быстро и удобно рисовать с помощью базовых инструментов и экспериментировать с цифровым творчеством без установки дополнительного ПО.

✨ Возможности
Кисть для свободного рисования
Выбор цвета
Настройка толщины линии
Инструменты для создания простых фигур
Очистка холста
Сохранение рисунка
🚀 Цель проекта
Создать простой и удобный инструмент для рисования в браузере, подходящий для быстрых набросков, идей и обучения основам работы с графикой на canvas.
